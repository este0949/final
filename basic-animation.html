02 Basic animation

In Basic Animation theme we started out with css animation. Using css transitions allowed us to change property values smoothly from one value to another.

Then we were introduced to a programming language Java script. It affects behaviour of the content we coded in html and styled in css. Before working on a group project, we had to animate our own character. Write a story with a convincing protagonist and antagonist and create that way a little story.

For the group project we used Java stript to glue everything together. We used Adobe Illustrator to illustrate backgrounds and characters. We could perfect our social skills, we had different type of responsibility and overall we could learn from one another.
